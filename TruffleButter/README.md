### Installation

    - Run testrpc or ganache on port 7545

    - Log into metamask

#### Clone the repo

    - git clone https://github.com/AvinashSingh786/TruffleButter && cd TruffleButter

#### Install a bunch of node modules and whatnot

    - npm install

#### Deploy the smart contracts

    - truffle migrate

#### Run dapp

    - ng serve

#### Test in browser

    - http://localhost:4200/

#### Test Contracts

    - truffle test
